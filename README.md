# react-modal-helper

> An utility to make the use of modals painless

[![NPM](https://img.shields.io/npm/v/react-modal-helper.svg)](https://www.npmjs.com/package/react-modal-helper) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save react-modal-helper
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'react-modal-helper'
import 'react-modal-helper/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
